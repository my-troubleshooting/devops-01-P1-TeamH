## 사용가능한 API 목록
1. 전체 상품조회 
2. 특정 상품조회
3. 상품 등록
4. 상품 삭제
5. 상품 내용 수정

## .env
dotenv를 사용하여 환경값으로 DB연결을 할것이기 때문에 
"ENDPOINT" 부분에 사용할 DB값을 주어야한다.

## npm run dev
To start the app in dev mode
Open Http://localhost:3000 to view it in the browser.

## npm start
To start production mode.

## npm run test
To start test cases.
